
# HSE Tracker

## Предисловие

В текущий момент проект находится на этапе активной разработки и уже доступен в телеграме

- [Telegram](https://t.me/hsetrackerbot)
- [Github организация проекта](https://github.com/hse-tracker)

## Суть приложения

- **Название проекта**: Сервис для отслеживания оценок студентов
- **Формат**: Telegram-бот + мини-приложение
- **Цель**: Создание инструмента для студентов ВШЭ, который позволит отслеживать свою успеваемость, планировать достижения по дисциплинам и получать уведомления о прогрессе.

Проект представляет собой сервис в формате Telegram-бота и мини-приложения внутри него, который помогает студентам отслеживать свои оценки и динамику успеваемости

## Используемые технологии

### Vue 3  
Основной фреймворк для разработки UI. В проекте используется подход Composition API, который позволяет удобно группировать логику внутри компонентов. Если сравнивать Vue 3 с React, то Vue предлагает более удобную систему работы: фреймворк сам отслеживает изменения данных без необходимости вручную указывать зависимости (как это делается в хуках react). Кроме того, формат однофайловых компонентов Vue (где разметка и логика лежат в одном файле) делает структуру кода более понятной и структурированной (не нужно плодить миллион подпапок для каждого компонента)

### Vite
Инструмент для сборки проекта и локальный сервер для разработки. В отличие от Create React App, который при каждом запуске или изменении обрабатывает весь проект целиком, Vite использует нативные ES модули браузера. Благодаря этому сервер разработки запускается практически мгновенно, а изменения в коде моментально отображаются в браузере. Это делает процесс разработки гораздо быстрее и удобнее

### Tailwind CSS
Фреймворк для стилизации UI, основанный на небольших вспомогательных классах. По сравнению с ручным написанием CSS, где необходимо придумывать уникальные имена классов и постоянно переключаться между файлами стилей и компонентами, Tailwind позволяет описывать стили прямо в HTML тегах. Это ускоряет процесс верстки и избавляет проект от разрастания неиспользуемого CSS кода
